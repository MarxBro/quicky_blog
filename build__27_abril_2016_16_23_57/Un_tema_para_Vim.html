Un tema para Vim

Este es un intento de tema minimalista para vim.

Como uso mayoritariamente C y Perl y no soy muy fanático de los colorines, es muy sutil la coloración.

Agregué un cambio de fg para el formato POD (documentación de Perl).

Util

Para debuggear y agregar definiciones de colores, se puede usar algo así.

" Mostar a que grupo de sintaxis corresponde
" con shit + Ctrl + P
nmap <C-S-P> :call <SID>SynStack()<CR>
function! <SID>SynStack()
    if !exists("*synstack")
        return
    endif
    echo map(synstack(line('.'), col('.')),'synIDattr(v:val, "name")')
endfunc

Esta idea la saqué de un vimcast.

Se ve más o menos así:

En el ejemplo se ven las ideas principales (que responden a mis preferencias):

  • Los números de línea a la izquierda están en negro y casi no se ven.
  • Las líneas de código no cambian de color, sino que
  • Solamente las variables están en negrita.
  • La documentación no aparece como un comentario, sino que
  • (La documentación) aparece como código pero en otro color.
  • Los comentarios están invertidos respecto a los colores del código.
